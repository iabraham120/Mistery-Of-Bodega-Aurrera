# Mistery Of Bodega Aurrera
 This is my little game developed as a project for a programming course; it's made in C using the Allegro library.
The game is about an employee at Bodega Aurrera (Mexican Walmart XD) whose boss promises him a raise, but cancels it the next day and promises it again the following day. The goal is to get your raise back before the cycle repeats.
